# react-One
使用react+redux+react-router粗略模仿了ONE官网，使用nodejs模拟后台
# 线上地址
http://120.78.142.165:3000/#/index/home（移动端效果更佳）
# 安装
1.克隆项目地址并下载（需要安装Git）


git clone git@github.com:xiaoxiaoxian123/react-One.git

2.通过npm安装本地服务第三方依赖模块（需要安装Node.js）

npm install
# 运行程序
执行node指令运行backEnd.js


node backEnd
# 浏览器打开地址
http://120.78.142.165:3000/#/index/home（移动端效果更佳）
